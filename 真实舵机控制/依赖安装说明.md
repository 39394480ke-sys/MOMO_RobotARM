# 真实舵机依赖安装说明

dry-run 模式不需要真实舵机依赖，也不连接硬件。

真实连接硬件、读取舵机状态、写入舵机目标位置、运行标定程序，需要安装：

- `lerobot`
- `feetech-servo-sdk`
- `pyserial`
- `pyyaml`

## 推荐环境

请使用专用 mamba 环境，避免污染系统 Python：

```bash
mamba create -n momo_rebot python=3.11 -y
mamba activate momo_rebot
python -m pip install lerobot feetech-servo-sdk pyserial pyyaml
```

验证：

```bash
mamba run -n momo_rebot python - <<'PY'
from lerobot.motors import Motor, MotorNormMode
from lerobot.motors.feetech import FeetechMotorsBus
import serial, yaml
print("真实舵机依赖导入成功")
PY
```

## 依赖关系

```text
lerobot / feetech-servo-sdk
= 用来和飞特 Feetech 舵机通信的驱动库

标定程序_calibrate.py
= 使用这些驱动库连接舵机，读取/写入寄存器，生成 标定文件.json

真实控制器
= 每次连接时读取 标定文件.json，然后控制机械臂

dry-run：不需要真实依赖，不连接硬件
标定程序：需要真实依赖，需要连接硬件
真实控制：需要真实依赖，需要连接硬件
```

## Feetech / LeRobot 接口

真实硬件通信使用 Feetech / LeRobot 接口：

```python
from lerobot.motors import Motor, MotorNormMode
from lerobot.motors.feetech import FeetechMotorsBus

motors = {
    "j10": Motor(10, "sts3215", MotorNormMode.DEGREES),
    "j11": Motor(11, "sts3215", MotorNormMode.DEGREES),
    "j12": Motor(12, "sts3215", MotorNormMode.DEGREES),
    "j13": Motor(13, "sts3215", MotorNormMode.DEGREES),
    "j14": Motor(14, "sts3215", MotorNormMode.DEGREES),
    "j15": Motor(15, "sts3215", MotorNormMode.DEGREES),
    "gripper": Motor(16, "sts3215", MotorNormMode.DEGREES),
}

bus = FeetechMotorsBus(port=port, motors=motors)
bus.connect()
```

读取舵机：

```python
bus.read("Present_Position", joint_name, normalize=False)
bus.read("Present_Velocity", joint_name, normalize=False)
bus.read("Present_Current", joint_name, normalize=False)
bus.read("Moving", joint_name, normalize=False)
```

写目标位置：

```python
bus.write("Goal_Position", joint_name, goal_raw, normalize=False)
```

写标定相关寄存器：

```python
bus.write("Operating_Mode", joint_name, 0, normalize=False)
bus.write("Homing_Offset", joint_name, value, normalize=False)
bus.write("Min_Position_Limit", joint_name, value, normalize=False)
bus.write("Max_Position_Limit", joint_name, value, normalize=False)
bus.write("Phase", joint_name, 28, normalize=False)
```

读取硬件已有 calibration：

```python
bus.read_calibration()
```

关闭扭矩和断开：

```python
bus.disable_torque()
bus.disconnect()
```

如果 bus 支持：

```python
bus.sync_read("Present_Position", normalize=False)
```

则优先用 `sync_read`，否则逐个 `read`。

## 缺依赖时

dry-run 主程序仍然可以运行。  
真实控制或标定程序缺少依赖时，会给出中文提示，要求先进入 `momo_rebot` 环境安装依赖。
