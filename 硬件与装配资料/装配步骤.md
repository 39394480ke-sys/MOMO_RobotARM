# 装配步骤

本文是面向新手的装配流程。当前仓库没有完整实物照片和螺丝型号，涉及方向、孔位、螺丝规格的位置先标记为“待补照片”或“待确认”。

## 1. 检查所有零件

操作：
核对 BOM 中的机械结构件、舵机、夹爪、线材、电源、控制板和工具。

检查点：
- 6 个舵机都在，ID 规划为 1 到 6。
- J2/J3/J5 对应多圈关节。
- 结构件无明显破损。
- 电源规格待确认前不通电。

## 2. 检查 3D 打印件

操作：
检查打印件外观、孔位、边缘和受力区域。

检查点：
- 没有裂纹、翘边、断层。
- 舵机能放入安装位。
- 孔位不过紧也不过松。
- 强受力件待补照片。

## 3. 安装底座

操作：
安装底座结构件和底部支撑。

检查点：
- 底座能平稳放置。
- J1 安装面水平。
- 螺丝型号待确认，不强行拧入打印件。
- 待补底座装配照片。

## 4. 安装 J1

操作：
将 ID 1 舵机安装到底座旋转位置。

检查点：
- 舵机 ID 是 1。
- 关节 key 对应 `shoulder_pan`。
- J1 是单圈关节。
- 线材预留空间，不影响底座旋转。

## 5. 安装肩部结构和 J2

操作：
安装肩部结构件和 ID 2 舵机。

检查点：
- 舵机 ID 是 2。
- J2 是多圈关节，key 为 `shoulder_lift`。
- joint_scale 固定为 `-5.3`。
- 金属减速模组方向待补照片。

## 6. 安装肘部结构和 J3

操作：
安装大臂、肘部结构件和 ID 3 舵机。

检查点：
- 舵机 ID 是 3。
- J3 是多圈关节，key 为 `elbow_flex`。
- joint_scale 固定为 `5.6`。
- 肘部运动不会刮线或撞到结构件。

## 7. 安装腕部 J4 / J5

操作：
安装腕部俯仰和腕部旋转结构。

检查点：
- J4 舵机 ID 是 4，单圈，key 为 `wrist_flex`。
- J5 舵机 ID 是 5，多圈，key 为 `wrist_roll`。
- J4 joint_scale 固定为 `-1.0`。
- J5 joint_scale 固定为 `1.0`。
- 腕部线材不能被 J5 卷入。

## 8. 安装夹爪

操作：
安装夹爪结构件、连杆和 ID 6 舵机。

检查点：
- 舵机 ID 是 6。
- key 为 `gripper`。
- 夹爪开合顺畅，无卡死。
- range_min/range_max 后续通过标定确认。

## 9. 安装摄像头

操作：
将摄像头固定到计划位置。

检查点：
- 摄像头视野不被夹爪遮挡。
- USB 线不会干涉 J4/J5。
- 摄像头型号和固定方式待确认。

## 10. 走线

操作：
按 `接线记录.md` 规划电源线、舵机总线和 USB 线。

检查点：
- 舵机串联顺序清晰。
- 线材不过度拉紧。
- 多圈关节附近留有活动余量。

## 11. 固定线材

操作：
使用扎带或理线扣固定线材。

检查点：
- J2/J3/J5 动作时线材不缠绕。
- 腕部旋转时线材不被拉扯。
- 电源线和通信线接头不松动。

## 12. 上电前检查

操作：
执行 `安全检查清单.md` 的上电前部分。

检查点：
- 正负极已确认。
- 机械臂周围无障碍。
- 急停或断电手段可立即操作。
- 默认 dry-run。

## 13. 第一次通电

操作：
接入 USB，确认串口；再接入舵机电源。

检查点：
- 没有异味、发热、异常声音。
- 控制电脑能看到串口。
- 舵机没有持续堵转。

## 14. 第一次读取舵机状态

操作：
在阶段四目录运行读取状态脚本。

```bash
cd 真实舵机控制
python 测试脚本_test/01_读取舵机状态.py
```

检查点：
- 6 个舵机 ID 能按预期读取。
- J2/J3/J5 raw 值没有异常跳变。
- 失败时先看 `故障排查手册.md`。

## 15. 第一次 dry-run

操作：
先执行 dry-run 移动测试。

```bash
cd 真实舵机控制
python 测试脚本_test/04_dry_run移动测试.py
```

检查点：
- 不真实写入舵机。
- 输出的 joint_scale、目标 raw 和限位检查符合预期。
- connect 只读取标定，不代表重新标定。

## 16. 第一次单关节小幅移动

操作：
确认标定、安全检查、电源和急停后，才执行真实单关节小幅移动。

检查点：
- 首次动作小于 2 度。
- 一次只动一个关节。
- 手离开机械臂运动范围。
- 观察方向是否符合预期，异常立即断电。
