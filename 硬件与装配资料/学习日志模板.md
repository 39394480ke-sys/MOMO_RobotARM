# 学习日志模板

```text
日期：
阶段：
今天完成：
遇到的问题：
排查过程：
修改的文件：
测试命令：
测试结果：
学到的概念：
下一步：
备注：
```

## 示例记录项

| 字段 | 建议写法 |
|---|---|
| 阶段 | 例如“阶段十二：硬件与装配资料” |
| 今天完成 | 写具体文件、测试或装配动作 |
| 遇到的问题 | 写现象，不只写“失败” |
| 排查过程 | 写检查了哪些命令、接线、配置 |
| 测试结果 | dry-run、读取状态、单关节小幅移动等 |
| 学到的概念 | 如多圈 absolute raw、zero_present_raw、URDF link |
