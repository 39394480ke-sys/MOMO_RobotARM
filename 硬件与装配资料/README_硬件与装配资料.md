# 硬件与装配资料

本目录是阶段十二的资料包，用来把前面阶段的软件系统整理成可以装配、检查和维护的项目文档。

本项目参考第三方 SOARM MOCE 硬件资料。以下信息用于硬件兼容和安全调试，发布前需要核验原始资料的许可证、署名要求和再分发限制：

| 关节 | 内部 key | 中文名称 | 舵机 ID | 模式 |
|---|---|---|---:|---|
| J1 | shoulder_pan | 底座旋转 | 1 | 单圈 |
| J2 | shoulder_lift | 肩部抬升 | 2 | 多圈 |
| J3 | elbow_flex | 肘部弯曲 | 3 | 多圈 |
| J4 | wrist_flex | 腕部俯仰 | 4 | 单圈 |
| J5 | wrist_roll | 腕部旋转 | 5 | 多圈 |
| J6 | gripper | 夹爪 | 6 | 单圈/夹爪 |

多圈关节固定为 `J2 / J3 / J5`。joint_scales 固定为：

```yaml
shoulder_pan: 1.0
shoulder_lift: -5.3
elbow_flex: 5.6
wrist_flex: -1.0
wrist_roll: 1.0
```

协议参数固定为：

```text
RAW_COUNTS_PER_REV = 4096
MULTI_TURN_PHASE_VALUE = 28
MULTI_TURN_ABSOLUTE_RAW_LIMIT = 30719
POSITION_MODE_VALUE = 0
```

## 使用顺序

1. 先读 `发布资料/最终README.md`，理解项目边界。
2. 按 `硬件清单_BOM.md` 核对零件。
3. 按 `装配步骤.md` 装配，缺照片的位置先标记。
4. 按 `接线记录.md` 接线，上电前逐项执行 `安全检查清单.md`。
5. 先 dry-run，再读取舵机状态，再小角度单关节移动。
6. 只有在必要时运行 `标定流程.md` 里的重新标定程序。

默认原则：先仿真，后 dry-run，最后才是真实机械臂。
