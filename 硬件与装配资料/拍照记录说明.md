# 拍照记录说明

装配照片用于让别人复刻时理解结构方向、线材走向、舵机 ID 和标定姿态。照片统一放到：

```text
硬件与装配资料/图片/
```

建议补充以下照片：

| 序号 | 照片主题 | 用途 | 文件名建议 |
|---:|---|---|---|
| 1 | 整体外观 | 说明整机完成状态 | `01_整体外观.jpg` |
| 2 | 底座装配 | 说明 J1 安装方向 | `02_底座装配.jpg` |
| 3 | J2/J3 减速模组 | 说明肩部和肘部减速结构 | `03_J2_J3减速模组.jpg` |
| 4 | 腕部 | 说明 J4/J5 装配和线材余量 | `04_腕部.jpg` |
| 5 | 夹爪 | 说明 J6、连杆和指尖结构 | `05_夹爪.jpg` |
| 6 | 接线 | 说明舵机串联和控制板接口 | `06_接线.jpg` |
| 7 | 电源 | 说明电源规格、接口和极性 | `07_电源.jpg` |
| 8 | 摄像头 | 说明摄像头安装位置 | `08_摄像头.jpg` |
| 9 | 标定姿态 | 说明逻辑 0 度 home 姿态 | `09_标定姿态.jpg` |
| 10 | 安全测试现场 | 说明首次测试的安全区域 | `10_安全测试现场.jpg` |

照片要求：

- 能看清舵机 ID 或安装位置。
- 能看清线材走向。
- 关键孔位和螺丝位置尽量拍近景。
- 不要把真实 API Key、个人路径、隐私信息拍入照片。
