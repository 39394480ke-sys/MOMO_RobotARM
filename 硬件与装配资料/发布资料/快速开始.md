# 快速开始

## 路线 1：只跑仿真

如果采用编号目录命名：

```bash
cd 03_仿真控制系统
python 主程序_main.py
```

当前仓库实际目录名：

```bash
cd 仿真控制系统
python 主程序_main.py
```

这条路线不需要真实舵机，不需要外部舵机电源。

## 路线 2：跑 Web dry-run

如果采用编号目录命名：

```bash
cd 11_系统集成
python 一键启动.py --mode dry_run
```

当前仓库实际目录名：

```bash
cd 系统集成
python 一键启动.py --mode dry_run
```

dry-run 不真实写入舵机，适合测试 Web、动作链路、Agent 工具桥接和安全逻辑。

## 路线 3：真实机械臂

真实机械臂控制有风险。开始前必须完成：

- 硬件装配检查。
- 接线和电源极性检查。
- 标定文件检查。
- dry-run 测试。
- 急停或断电手段确认。
- 第一次动作小于 2 度。

如果采用编号目录命名：

```bash
cd 11_系统集成
python 依赖检查.py
python 标定检查.py
python 一键启动.py --mode real
```

当前仓库实际目录名：

```bash
cd 系统集成
python 依赖检查.py
python 标定检查.py
python 一键启动.py --mode real
```

如果标定检查失败，不要进入 real 模式。connect 只读取标定文件，不等于重新标定。
