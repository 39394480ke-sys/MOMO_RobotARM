# 安全检查清单

真实机械臂控制有风险。默认 dry-run，只有完成标定、供电、接线和环境检查后才允许真实移动。

## 1. 上电前

- [ ] 机械臂周围无障碍物。
- [ ] 手、工具、线材不在运动路径内。
- [ ] 舵机电源正负极已确认。
- [ ] 电源参数已根据硬件确认。
- [ ] 线材没有缠绕 J2/J3/J5 多圈关节。
- [ ] 急停或断电手段可立即操作。
- [ ] 没有让夹爪夹住硬物。

## 2. 连接前

- [ ] 串口正确：Mac 常见 `/dev/tty.usbmodemXXXX`，开发板固定使用 `/dev/momo-servo`。
- [ ] 舵机 ID 与 `舵机ID与关节对应表.md` 一致。
- [ ] 标定文件存在。
- [ ] connect 只读取标定，不等于重新标定。
- [ ] Web、GUI、Agent 没有多人同时控制硬件。

## 3. 标定前

- [ ] 明确是否真的需要重新标定。
- [ ] J2/J3/J5 是多圈，必须使用 phase=28。
- [ ] J1/J4/夹爪是单圈/夹爪，需要确认 range_min/range_max。
- [ ] 机械臂处于可手动调整的安全姿态。
- [ ] 不在负载状态下标定。

## 4. dry-run 前

- [ ] `dry_run` 默认开启。
- [ ] 先运行依赖检查和标定检查。
- [ ] dry-run 输出的关节 key、ID、joint_scale 正确。
- [ ] 不直接写 raw。

## 5. 第一次真机移动前

- [ ] 已通过 `系统集成/标定检查.py`。
- [ ] 已确认电源稳定。
- [ ] 已读取舵机状态。
- [ ] 首次动作不超过 0.5 度。
- [ ] 一次只移动一个关节。
- [ ] 操作者手离开机械臂运动范围。

## 6. 动作回放前

- [ ] 动作文件来源可信。
- [ ] 动作已经在仿真或 dry-run 中检查。
- [ ] 动作速度和幅度适合真实机械臂。
- [ ] 夹爪动作不会长时间堵转。
- [ ] 周围无人靠近。

## 7. 语音 Agent 控制前

- [ ] Agent 不直接写舵机 raw。
- [ ] Agent 只通过阶段八 Web API 间接控制。
- [ ] 默认 dry-run。
- [ ] 真实模式需要人工确认。
- [ ] Agent 工具调用受到安全策略限制。

## 8. 断电前

- [ ] 停止 Web、GUI、视觉、Agent 等控制入口。
- [ ] 机械臂停止运动。
- [ ] 先让夹爪释放负载。
- [ ] 记录异常声音、掉线或过热情况。
- [ ] 关闭舵机电源，再整理线材。
