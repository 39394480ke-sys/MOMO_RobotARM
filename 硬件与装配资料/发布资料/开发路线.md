# 开发路线

## 阶段一：项目总览

目标：明确项目目标、边界和阶段安排。

完成标准：能说明项目要做什么、不做什么、如何分阶段。

关键文件：`项目整体结构.md`。

学到的概念：项目拆解、模块边界、学习型工程实践。

## 阶段二：机械臂基础概念

目标：理解关节、坐标、舵机、标定、安全规则。

完成标准：能解释 J1-J6 和真实机械臂风险。

关键文件：`机械臂基本概念/关节定义.md`，`机械臂基本概念/安全规则.md`。

学到的概念：关节、自由度、raw、标定、安全动作。

## 阶段三：仿真控制系统

目标：先在软件中跑通机械臂逻辑。

完成标准：能加载配置、控制仿真关节、播放动作。

关键文件：`仿真控制系统/主程序_main.py`，`仿真控制系统/机械臂模型_robot_arm.py`。

学到的概念：关节状态、姿态库、动作播放。

## 阶段四：真实舵机控制系统

目标：接入 Feetech 舵机、标定和 dry-run。

完成标准：能读取舵机状态、dry-run 映射、真实模式前检查。

关键文件：`真实舵机控制/真实机械臂控制器_real_arm_controller.py`，`真实舵机控制/标定程序_calibrate.py`。

学到的概念：`zero_present_raw`、`home_present_raw`、多圈 absolute raw、joint_scales。

## 阶段五：URDF / 运动学 / 3D 仿真

目标：建立机器人结构描述和运动学能力。

完成标准：能检查 URDF、运行 FK/IK、显示 3D 模型。

关键文件：`URDF运动学仿真/urdf/soarmoce_urdf.urdf`，`URDF运动学仿真/运动学模型_kinematics_model.py`。

学到的概念：URDF、link、joint、mesh、正逆运动学。

## 阶段六：动作录制与回放增强

目标：把动作变成可保存、可检查、可回放的序列。

完成标准：能录制动作、插值、回放、检查多圈关节。

关键文件：`动作录制与回放增强/动作录制器_action_recorder.py`，`动作录制与回放增强/动作回放器_sequence_player.py`。

学到的概念：动作帧、插值、安全摘要、回放确认。

## 阶段七：GUI 图形界面

目标：用桌面界面操作机械臂和查看状态。

完成标准：GUI 能启动，能进行 dry-run 控制和查看标定状态。

关键文件：`GUI图形界面/GUI主程序_main.py`，`GUI图形界面/gui_app/主窗口_main_window.py`。

学到的概念：控制器桥接、页面状态、安全确认。

## 阶段八：Web 控制台 / Quick Control API

目标：提供 Web API 和前端控制台。

完成标准：API health 正常，能 dry-run 关节控制和动作接口。

关键文件：`Web控制台/启动Web服务.py`，`Web控制台/backend/app.py`。

学到的概念：HTTP API、WebSocket、状态管理、远程控制边界。

## 阶段九：视觉识别与跟随

目标：接入摄像头、检测目标并生成跟随命令。

完成标准：摄像头预览正常，视觉服务能 dry-run 跟随。

关键文件：`视觉识别与跟随/视觉主程序_main.py`，`视觉识别与跟随/vision/视觉跟随_controller.py`。

学到的概念：摄像头源、目标偏移、平滑滤波、视觉闭环。

## 阶段十：语音 Agent

目标：用语音或文本驱动安全受限的工具调用。

完成标准：Agent 能通过工具桥接执行 dry-run 控制，不直接写 raw。

关键文件：`语音Agent/语音Agent主程序_main.py`，`语音Agent/agent/安全策略_safety_policy.py`。

学到的概念：STT、TTS、工具调用、Agent 安全边界。

## 阶段十一：系统集成 / 一键启动

目标：统一启动、停止、健康检查、日志和配置。

完成标准：能运行 dry-run 全链路，real 模式有标定和安全门槛。

关键文件：`系统集成/一键启动.py`，`系统集成/健康检查.py`，`系统集成/标定检查.py`。

学到的概念：服务编排、模式管理、健康检查、统一日志。

## 阶段十二：硬件与装配资料 / 项目文档收尾

目标：形成别人能理解、能装配、能排查、能继续开发的资料包。

完成标准：BOM、装配、接线、标定、安全、排障、最终 README 和发布清单完整。

关键文件：`硬件与装配资料/README_硬件与装配资料.md`，`硬件与装配资料/发布资料/最终README.md`。

学到的概念：开源硬件文档、装配维护资料包、发布检查、维护记录。
